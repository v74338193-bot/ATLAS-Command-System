The project is designed to simplify the process of monitoring and controlling one or multiple Windows devices through an easy-to-use Telegram interface, while maintaining secure and authorized access.

Key Features
System Monitoring: CPU, RAM, battery status, uptime, running processes.
Files & Directories: Browse, download, create, delete files.
Visual Devices: Screenshots and camera capture.
Network Management: Local & public IP, saved WiFi passwords.
Alerts & Notifications: Telegram messages or Windows pop-ups.
Program Execution: Run & stop Python scripts or executables.
Access Protection: Owner ID authorization.
Flexibility: Multi-device support via one bot.
Usage
Run on a Windows device.HIRA9L.py
Send from the authorized Telegram account./start
Control and monitor the device using Telegram commands.
